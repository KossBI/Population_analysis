# Анализ стандартизованных коэффициентов смертности

## Описание проекта

Пайплайны на Python для сравнения демографических показателей на примере смертности по трём муниципалитетам за семь лет для четырёх причин смерти. Рассчитаны стандартизованные коэффициенты смертности (СКС) на 100 000 населения в следующих разрезах:

- суммарный (все причины, оба пола)
- отдельно для мужчин и отдельно для женщин
- отдельно по четырём причинам смерти

Выполнены:
- попарные сравнения СКС между районами по полу и причинам смерти
- попарные сравнения СКС внутри районов между полами и причинами смерти
- проверка на наличие временных трендов по полу и причинам смерти (только в Анализ.ipynb)
- рассчёт коэффициентов корреляции Пирсона и Спирмена для значений общей (без учёта возрастных групп) доли умерших и СКС по годам и полам между муниципалитетами и статистических уровней значимости этих коэффициентов

Построены:
- половозрастные пирамиды за период и по годам для всех муниципалитетов
- барплоты для общей (без учёта возрастных групп) доли умерших и СКС по годам для всех муниципалитетов

## Формат входных данных

Таблица csv.

Колонки:
- Год: int
- Муниципалитет: str
- Пол: str
- Возрастная группа: str
- Причина смерти: str
- Стандартный вес группы: str (преобразуется в пайплайне во float одной строчкой в разделе "Приведение входных данных к удобному виду")
- Число умерших: int
- Численность населения: int

## Как воспроизвести анализ

1. Скачать репозиторий (кнопка Code → Download ZIP) и распаковать

2. Установить зависимости:
   `pip install -r requirements.txt`

### Для Анализ.ipynb

- Запустить Jupyter Notebook:
   `jupyter notebook путь/до/папки/Анализ.ipynb`

- В открывшемся ноутбуке выполнить все ячейки (Cell → Run All)

### Для Анализ.py

- Запустить скрипт:
   `python путь/до/папки/Анализ.py`

## Методы расчёта

- Прямая стандартизация (возрастные веса переводятся из процентов в доли).
- Нормальное приближение распределения Пуассона для границ доверительного интервала если число смертей D в выборке D > 25.
- Метод Добсона для рассчёта границ доверительного интервала если число смертей D в выборке 10 ≤ D ≤ 25.
- Утверждается отсутствие различия между СКС на основе перегрытия доверительных интервалов.
- Взвешенная линейная регрессия для проверки наличия временной динамики.

Оптимальность используемых для выбора метода расчёта границ ДИ диапазонов D показана в работе Морриса и соавторов:
[PubMed Central](https://pmc.ncbi.nlm.nih.gov/articles/PMC6303975/)

## Результаты

Ноутбук содержит пять итоговых таблиц:

1. df_result — численность населения, СКС, границы ДИ СКС и статистическая значимость отличия СКС от нуля для всех разрезов
2. slices_d2d_df — сравнение между муниципалитетами
3. df_pair_in_district — сравнение внутри муниципалитетов
4. slices_all_years_d2d_df — сравнение между муниципалитетами с усреднением по всему временному диапазону
5. df_pair_all_years_in_district — сравнение внутри муниципалитетов с усреднением по всему временному диапазону

Выводы по результатам анализа:
1. Районы 1 и 2 не отличаются как по уровню смертности так и по её структуре.
2. В Районе 3 женская смертность и смертность от болезней сердца ниже, чем в Районах 1 и 2.
3. В Районах 1, 2 и 3 смертность от болезней сердца превышает смертность от остальных причин.
4. Есть статистически значимый тренд на снижение смертности от болезней сердца в Районе 3.
5. Есть статистически значимая корреляция значений общей доли умерших и СКС для мужчин между Районами 1 и 2.

## Уровень значимости

Во всех статистических тестах используется α = 0.05 (используется переменная alpha).
