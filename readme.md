# Анализ стандартизованных коэффициентов смертности

## Описание проекта

Пайплайн на python для сравнения демографических показателей методом прямой стандартизации на примере смертности по трём муниципалитетам за семь лет для четырёх причин смерти. Рассчитаны стандартизованные коэффициенты смертности (СКС) на 100 000 населения в следующих разрезах:

- суммарный (все причины, оба пола)
- отдельно для мужчин и женщин
- отдельно по четырём причинам смерти

Выполнены:
- попарные сравнения СКС между районами, полами и причинами смерти (z-тест)
- анализ временных трендов (взвешенная линейная регрессия)

## Требования

- Python 3.11 или выше
- Библиотеки из файла requirements.txt

## Как воспроизвести анализ

1. Скачать репозиторий (кнопка Code → Download ZIP) и распаковать

2. Установить зависимости:
   `pip install -r requirements.txt`

3. Запустить Jupyter Notebook:
   `jupyter notebook Анализ.ipynb`

4. В открывшемся ноутбуке выполнить все ячейки (Cell → Run All)

## Файлы в репозитории

- Анализ.ipynb — основной код анализа
- Тестовое _ Данные - Лист1.csv — исходные данные
- requirements.txt — список зависимостей
- README.md — описание проекта

## Методы расчёта

- Прямая стандартизация (возрастные веса переводятся из процентов в доли)
- Пуассоновская аппроксимация для стандартной ошибки
- Z-тест для попарного сравнения СКС
- Взвешенная линейная регрессия (веса = 1/SE²)

## Результаты

Ноутбук содержит пять итоговых таблиц:

1. df_result — СКС, стандартные ошибки СКС и статистическая значимость отличия СКС от нуля для всех разрезов
2. df_result_comparison_out — сравнение между муниципалитетами
3. df_result_comparison_in_gender — сравнение между полами
4. df_result_comparison_in_death_cause — сравнение между причинами смерти
5. df_result_regression — временные тренды

## Уровень значимости

Во всех статистических тестах используется α = 0.05 (используется переменная SL)
